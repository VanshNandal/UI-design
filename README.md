# React SPA Demo

This project demonstrates a responsive Single Page Application using React, Bootstrap, and Material UI.

## Features
- Responsive navigation bar (Bootstrap)
- Bootstrap form and card-based layout
- Material UI modern form

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

---

**Software Requirements:**
- Node.js
- Web Browser

**Component Libraries:**
- Bootstrap
- Material UI (@mui/material)
